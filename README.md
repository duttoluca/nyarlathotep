nyarlathotep
===

Django Website.

Partito come semplice blog, è un esercizio/studio sul framework Django.

Utilizza al momento:
django "base"
south
django debug toolbar
django comments
django markup
django flatpages
django admin
Twitter Bootstrap

Per utilizzarlo:
- creare un virtualenv con python 2.7.3
- pip install -r requirement.txt
- git clone git@github.com:duttoluca/nyarlathotep.git
